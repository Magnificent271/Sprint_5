## Удалось реализовать следующие тесты:
1. `test_add_new_book` - Проверка добавления новой книги в словарь books_genre
2. `test_add_new_book_invalid_length` - Проверка валидации на количество символов в названии книги. Количество символов 0 и 42
3. `test_set_book_genre_add_genre` - #Проверка установки жанра книги. Позитивный кейс. Жанр есть в списке. Негативный кейс. Жанра нет в списке
4. `test_set_book_genre_not_book_in_list` - Негативный кейс. Установка жанра не существующей книге
5. `test_get_book_genre` - Проверка корректного возвращения жанра для указанной книги
6. `test_get_books_with_specific_genre` - Проверка получения списка книг с определенным жанром
7. `test_get_books_for_children` - Проверка возвращения списка книг доступным детям
8. `test_add_book_in_favorites` - Проверка добавления книги в Избранное
9. `test_add_book_in_favorites_twice` - Проверка добавления одной и той же книги в Избранное
10. `test_delete_book_from_favorites` - Проверка удаления книги из Избранного
11. `test_get_list_of_favorites_books` - Проверка получения списка Избранных книг

### Для запуска тестов использовать команду - `pytest -v tests.py`
